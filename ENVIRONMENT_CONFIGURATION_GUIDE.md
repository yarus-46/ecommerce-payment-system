# Environment Configuration Guide

## Required Environment Variables

- APP_NAME
- APP_ENV
- APP_KEY
- APP_DEBUG
- APP_URL
- DB_CONNECTION
- DB_HOST
- DB_PORT
- DB_DATABASE
- DB_USERNAME
- DB_PASSWORD

## Setup Steps

1. Copy `.env.example` to `.env`
2. Update database credentials
3. Generate application key
4. Run database migrations
5. Start the development server
